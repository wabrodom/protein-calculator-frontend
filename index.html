Protein Calculator

คุณอาจจะพอทราบว่า คนเราควรกินโปรตีน 1 กรัม ต่อน้ำหนักตัว 1 กิโลกรัม แต่ก็อาจจะเพิกเฉย หรือไม่ใส่ใจว่าจริงๆแล้ว อาหารที่กินมีโปรตีนเท่าไร น้ำหนักอาหารที่กินไม่ใช่น้ำหนักโปรตีนล้วนๆ เช่น ไข่ 1 ฟอง มีโปรตีนจริง 12% นอกนั้นส่วนใหญ่เป็นน้ำ

บางคนอาจยังไม่ทราบว่าในข้าวก็มีโปรตีน ข้าวสวย 1 ถ้วยมีโปรตีนประมาณ 4-5 กรัม ผู้ที่กินแป้งน้อยอยู่แล้วหรือผู้ทำ IF จึงมีโอกาสขาดโปรตีนกว่าคนที่กิน 3 มื้อ แต่จะแน่ใจได้อย่างไร ต่อให้คุณกิน 3 มื้อ คุณก็อาจกินโปรตีนไม่ถีง

จากที่ผมอ่านบทความเกี่ยวกับปริมาณโปรตีนแนะนำของกระทรวงสาธารณสุขของประเทศ สหรัฐอเมริกา(NIH)แนะนำ หรือRDA เท่ากับ 0.8 g/kg หลายบทความก็ชี้ชัดว่า 0.8 คือขั้นต่ำ หรือจะตีความได้ว่า ถ้าคุณกินต่ำกว่า 0.8 โดยเฉลี่ยทุกวันๆ เป็นเวลานาน ร่างกายคุณจะเริ่มป่วย เมื่อมานั่งขบคิดพิจารณากับตัวเอง ผมก็คิดได้ ว่าตัวเองกินโปรตีนไมถึง ต่อให้กินไข่วันละ 4 ฟอง ถ้าน้ำหนัก 50 ใช้เกณฑ์แบบมาตรฐานคือ 1 กรัม/กก. ก็ได้โปรตีนแค่ครึ่งเดียวเท่านั้น เป็นความจริงที่เบิกเนตรผมมาก จนนำมาสู่การทำโปรเจคเล็กๆนี้ และผมจะทยอยอัพเดทไฟล์ JSON บน server ต่อไป

ผมหวังว่า web app นี้จะเป็นประโยชน์ และเป็นไอเดียให้ทุกคนนำไปปรับใช้ได้
